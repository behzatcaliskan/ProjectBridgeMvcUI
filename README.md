# Project Bridge MVC UI

Bu proje, MVC tabanlı bir kullanıcı arayüzü çalışmasıdır. HTML, CSS, JavaScript ve Bootstrap kullanılarak responsive web sayfaları oluşturulmuştur.

## Kullanılan Teknolojiler

- ASP.NET Core MVC
- HTML
- CSS
- JavaScript
- Bootstrap

## Proje Özellikleri

- Responsive kullanıcı arayüzü
- Sayfa yönlendirme yapısı
- MVC view yapısı
- Statik dosya yönetimi
- Template düzenleme ve uyarlama

## Proje Amacı

Bu proje, ASP.NET Core MVC tarafında kullanıcı arayüzü geliştirme, hazır template düzenleme ve frontend dosya yapısını projeye entegre etme becerisini geliştirmek amacıyla hazırlanmıştır.

## Teknik Kazanımlar

- MVC view yapısı
- Layout kullanımı
- CSS ve JavaScript dosyalarının projeye bağlanması
- Responsive tasarım mantığı
- Kullanıcı arayüzü düzenleme
